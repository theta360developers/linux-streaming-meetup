Building libuvc-theta and gstreamer sample program on Ubuntu 20.04 Focal Fossa.

Aug 20, 2020

  1. Grab the patched version of libuvc from GitHub.
  2. Clone libuvc-theta-sample from GitHub
  3. build and install libuvc. Make sure you build the patched branch.
  4. build the sample app
  5. Plug RICOH THETA into Linux USB port. Put into live mode with the mode button on the side of the camera or with the USB API for StillCaptureMode which you need to set to Live streaming with value 0x8005.
